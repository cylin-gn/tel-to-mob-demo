Overview

What is the task from Telecom to Mobility?

A new task that leverages undirected features (Geographical Cellular Traffic (GCT) flows) to predict directional features (Mobility Flows).

This task advances a new graph modeling paradigm: using historical undirected features to predict directional features, which differs from the current prediction problems that focus on the same domain.

Download Data

Dataset: TeltoMob

GCT flows

GCT (Geographical Cellular Traffic): Refers cellular traffic data associated with its generated location.
GCT flow: Refers to the accumulated GCT within a region over a specific time interval (typically 15 minutes). Over time, the series of GCT flows reflect the patterns of the respective region.
GCT Flow Examples

Mobility Flow

GCT pairings: Refers to the association of consecutive GCT entries for a single user when they are found in adjacent road segments within a 15-minute timeframe.
Mobility flow: The total count of GCT pairings for a road route, recorded at fixed time intervals (e.g., 15 minutes).
